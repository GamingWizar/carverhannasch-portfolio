# Portfolio

This is my portfolio website built from scratch. It was created using TypeScript and React, and is hosted on a Google Cloud Virtual Machine using a custom domain.
Visit it at https://carverhannasch.com

## Project Features

- Semantic HTML5
- Flexbox
- Grid
- CSS
- Flat BEM
- React
- TypeScript
- Web Hosting
